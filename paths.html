Web Developer

Build websites from Zero → Real projects → Portfolio.

Start this path →

Step 1 — Foundations

  • HTML structure (headings, links, images)
  • CSS basics (colors, spacing, box model)
  • Responsive basics (mobile-friendly)
Deliverable: A clean personal landing page.

Step 2 — Core Skills

  • JavaScript basics (variables, conditions, loops)
  • DOM & events (click, form, scroll)
  • GitHub basics (publish your work)
Deliverable: Interactive page (FAQ / Contact validation).

Step 3 — Real Projects

  • Build 3–5 projects (responsive + interactive)
  • Design system (consistent UI)
  • Deploy on Netlify (live links)
Deliverable: Portfolio website with 3 projects.
Goal: Not memorizing code — learning how to build & solve problems.

What You Need To Start

English

You don’t need to be perfect. But you should understand basic words like: button, link, submit, error.

Math

Not advanced math. You mainly need logic: steps, conditions, and problem-solving.

Mindset

Build → break → fix → improve. Mistakes are part of the process. That’s how real developers learn.
